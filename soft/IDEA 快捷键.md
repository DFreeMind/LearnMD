# Mac

`command + n` ：快速生成类的构造器

`command + alt + v`：快速生成变量

`ctrl + I`：快速实现类实现接口的方法



## 缩写快捷键

`psvm`：快速生成 `main`函数

`sout`：快速生成 `System.out.println`

`iter`：    Iterate (for each..in) 
`itin`：    Iterate (for..in) 
`itli`：    Iterate over a List
`itar`：    Iterate elements of array 
`ritar`：  Iterate elements of array in reverse order
